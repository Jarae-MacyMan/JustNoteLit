Just Note Lit
=============

A full stack CRUD web application that allows users to take and update notes 


Run Locally
===========

Clone the repo!


Server
======

1. ``cd JustNoteIt`` 
2. Make sure you have downloaded python and activated a virtual environment
3. ``source venv/bin/activate``
4. ``pipenv install``
5. ``python main.py``


Client
======
1. ``cd justnotelit``
2. ``npm install``
3. ``npm start``
